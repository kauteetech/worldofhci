World of Hyper-Converged Infrastructure



What is Hyper-Converged Infrastructure ? Let's keep it simple, Hyper-Converged Infrastructure is nothing but running core data center infrastructure components like compute, network, storage and security services on a commodity x86 physical server. All the core data center services (Compute, Network, Storage and Security) run as software services.
What is WoHCI (World of Hyperconverged Infrastructure) ? WoHCI accessed through www.worldofhci.com is a web site with a web application running on it. This web application can be used to get an idea on the product and its features based on various selection criterias. The ratings provided or based on my understanding of given solution. see "Why HCI was created " section for more operational aspects of the application.
Why WoHCI was created ?

Hyperconverged Infrastructure has already gained it's momentum and are being evaluated or considered for most of the data center infrastructure requirements. As we know whenever there is something gaining attraction, every one wants to participate and gain whatever they can. Almost all of the Data center infrastructure vendors today have HCI offerings.

These HCI offerings by the vendors are either built from grounds up or rebranding of current products or through new acquisitions. Every vendors want say the HCI story with their offerings and this story most part is not consistent and creates confusion among customers. Not all customers have the time and skill to go through every vendors offering in-depth due to time and other constraints.

WoHCI is focused around bringing in clarity and simplify the process of evaluation of HCI solution. WoHCI is a simple tool using which customers can understand various vendors stand or capability, based on customers choice of functions or metrics.

This tool uses a two step process.
  1. Select one or more vendors from the list of avilable vendors
  2. Select one or more functions or metrics from the list of available functions

On selection of functions and vendors, every vendor has score based on the functions selected. The vendors with high score are the preferred ones based on the selected functions. Customers can pick or drop vendors or functions at any point in time. The application will change the score fornthe selected vendors in real time.
